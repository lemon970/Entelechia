# 隐德来希全牌库平衡审查进度

日期：2026-07-10

## 当前决定

- 本轮整体以加强为主，只修复实现错误、空收益分支、意外自杀和低成本无限循环等明确问题。
- 心烛受能量、牌序和敌方生命限制，sim2 的高生命木桩理论值不直接作为削弱依据；本轮不主动回调心烛数值。
- 高血状态定义为 `CurrentHp > MaxHp * 0.5`，主要提供过牌、加费和主动压血能力。
- 低血状态定义为 `CurrentHp <= MaxHp * 0.5`，主要提供回复、格挡、返能和爆发兑现能力。
- 恰好半血必须进入低血分支，不能出现付费后没有附加收益的死区。
- 生命代价继续经过统一支付检查：通常至少保留 1 点生命；拥有不死血统或血魔重盈等明确濒死保护时允许支付致死代价。

## 首批候选方向

| 卡牌 | 候选方向 | 主要目的 |
|---|---|---|
| 血愈 | 高血提高抽牌；低血保留过牌并回复生命 | 建立双状态基础牌模板 |
| 自噬 | 提高能量收益；高血可附带少量过牌 | 解决前期能量不足和主动压血困难 |
| 血色重构 | 高血压到半血并获得明显资源；低血回复到半血并获得防守/萃血收益 | 稀有双状态锚点 |
| 赤色余温 | 低血时无需本回合先失血也能获得完整防守收益 | 提高低血续航可靠性 |
| 借血 | 高血时提高返能稳定性 | 提供主动压血与过牌启动 |
| 血色脉冲 / 逆流斩 | 将低血状态纳入返能条件，但必须检查低成本循环 | 低血持续行动力 |
