# 隐德来希牌库平衡第七批报告

日期：2026-07-11

## 本批范围

- 吸血（BloodDrain）
- 深红壁垒（CrimsonShield）
- 深红鞭痕（CrimsonLash）
- 切创（Lacerate）
- 血爪连斩（BloodClawSlash）

## 审查结论

- 吸血原本在攻击后施加萃血，牌名对应的续航无法由本牌即时兑现，高血也缺少轮转价值。
- 深红壁垒的 `8/11` 格挡低于无代价的血盾 `10/13`，自身失血没有换来足够收益。
- 深红鞭痕被同费同稀有度的切创严格压过，两张牌缺少选牌差异。
- 血爪连斩原总伤害仅 `8/12`，所在血速多段构筑在高压场景仍偏弱。
- 本批没有加入新的返能来源；高血过牌和低血爆发处于互斥生命区间，避免把加强转化为廉价无限。

## 轮子与 API 复核

- 抽牌复用 `EntelechiaCard.DrawCards`。
- 条件格挡复用 `CreatureCmd.GainBlock`。
- 多段攻击复用项目当前版本兼容的 `ExecuteCardAttack` / `AttackCommand.WithHitCount`，不使用 BaseLib 旧版 `CommonActions.CardAttack`。
- 切创的额外生命流失复用 `TurnStateTracker.LoseHpTracking` 和游戏原生 `DamageProps.nonCardHpLoss`，不计作额外攻击命中。
- 高低血统一复用 `IsLowHealth`，边界仍为 `CurrentHp <= MaxHp * 0.5m`。

## 实装调整

### 吸血

- 伤害保持 `8/11`。
- 高血：攻击后施加 `2/3` 层萃血并抽 1 张牌。
- 低血：攻击前施加 `1/2` 层萃血，使本牌可即时触发萃血；不抽牌。
- 死亡目标不继续施加状态或抽牌。

### 深红壁垒

- 基础格挡从 `8/11` 提高到 `10/13`。
- 高血：保留对自身施加 1 层失血，用于主动压血和后续失血联动。
- 低血：不再施加自身失血，额外获得 2 点格挡，总计 `12/15`。

### 深红鞭痕

- 伤害从 `6/9` 提高到 `7/10`。
- 高血：施加 `2/3` 层失血并抽 1 张牌，定位为轮转型失血入口。
- 低血：不抽牌，改为施加 `3/4` 层失血。

### 切创

- 伤害从 `7/10` 提高到 `8/11`，失血保持 `3/4`。
- 低血时额外使存活目标失去 `2/3` 点生命；该段不是攻击伤害，不增加攻击命中数。
- 低血即时总输出为 `10/14`，随后仍保留失血压力。

### 血爪连斩

- 单段伤害从 `2/3` 提高到 `3/4`，仍攻击 4 次，总伤害从 `8/12` 提高到 `12/16`。
- 不增加命中次数、抽牌或返能，保留力量、多段和萃血消费定位。

## sim2 结果

- 血速多段构筑，12 HP、每回合 12 敌伤：平均伤害约 `15.66 -> 16.87`，死亡率维持 `40%`。
- 同场景平均出牌 `3.53`、平均获得能量 `0.35`、上限命中率 `0%`，加强来自单牌伤害，不是行动力膨胀。
- 26 HP 高压死亡率 `5%`，25 HP 高压死亡率 `0%`，未出现半血边界异常。
- 其他现有构筑不包含本批其余四张牌，矩阵数值保持第六批基线；其具体分支由 targeted tests 验证。

## 结论

第七批消除了两组同费牌的严格支配关系：吸血承担高血轮转与低血即时续航，深红鞭痕承担高血铺状态轮转，切创承担低血即时爆发，深红壁垒承担高血主动压血与低血防守。血爪连斩提高了偏弱多段体系的平均伤害，同时没有增加能量或出牌次数。结果符合“低血续航和爆发更强、Mod 角色可略高于原版”的方向。
