# 隐德来希全牌库平衡设计

日期：2026-07-10

## 目标

重新审查 57 张实际卡牌和直接关联能力，形成以加强为主的统一数值路线：

- 高血阶段拥有足够的过牌、加费和主动压血能力，不再经常因为能量或牌序无法启动。
- 低血阶段拥有可靠续航、格挡、返能和高爆发，允许承担比原版角色更高的平均输出。
- 心烛继续作为高生命敌人的延迟真伤池，不因为 sim2 木桩理论值高而主动削弱。
- 修复半血空收益、描述与实现不一致、生命代价绕过保护和明显低效率牌。
- 不改卡牌名称、ID 和图片资源。

## 非目标

- 不把每张牌都改成同时具有高血与低血分支。
- 不追求与原版角色相同的均值；Mod 角色可以略强。
- 不用单一伤害/能量指标替代生存、牌序和资源机会成本。
- 不为压低理论爆发而回调已经通过实战方向确认的心烛生成倍率。

## 统一生命状态

```text
高血：CurrentHp > MaxHp * 0.5
低血：CurrentHp <= MaxHp * 0.5
```

实现时在 `EntelechiaCard` 提供统一判断方法，所有卡牌共用同一比较规则。恰好半血属于低血，避免 `BloodRebuild` 当前 `>` / `<` 写法产生空分支。

低血条件在卡牌开始结算时快照。若一张牌先回复到半血以上，仍按打出前状态获得低血收益；否则赤色余温等牌会因为自身回复而丢失附加效果。

## 资源设计

### 高血阶段

高血资源牌应至少完成以下一种工作：

- 净增加可用能量。
- 提高本回合抽牌或出牌数。
- 主动把生命压到低血区。
- 把当前不需要的萃血回复资源转为过牌或伤害。

高血牌的主要代价是生命、消耗、条件资源或未来回合风险，不应再同时支付高牌费并只得到延迟的小收益。

### 低血阶段

低血收益应分散到不同牌上：

- 直接回复：血愈、赤色余温、血债清算、断魂采血。
- 格挡：缝合、赤色余温、凝血屏障、凝血本能。
- 行动力：血色脉冲、逆流斩、借血的条件返能。
- 濒死保护：不死的血统、血魔重盈。
- 爆发兑现：惜别终幕、暴血乱舞、血速多段、心烛。

回复按有效回复衡量，溢出治疗不算强度。低血返能不能仅凭一张普通牌形成无成本无限循环。

## 已确认改动

以下结论已经吸收完整 Uncommon 审查和 Rare/Power 审查。除明确写为“维持”的牌外，均进入实施计划；数值验收仍以 `docs/superpowers/reviews/2026-07-10-entelechia-sim2-balance-matrix.md` 为准。

### 基础资源牌

#### 血愈

保持 1 费。

- 高血：抽 2 / 3 张牌。
- 低血：回复 5 / 7 点生命，抽 1 / 2 张牌。

它成为最直观的双状态普通牌：高血压缩牌序，低血提供续航但不同时获得最高抽牌量。

#### 自噬

保持 0 费和消耗。

- 失去 3 / 2 点生命。
- 获得 2 点能量。

生命支付应在消耗前完成。它负责前期主动压血和能量启动，不额外抽牌，避免一张普通牌同时包办全部资源。

#### 血色重构

保持 2 费，升级后 1 费，消耗。

- 高血：将生命调整到 50%，获得 3 点能量并抽 1 张牌。
- 低血：回复到 50% 生命，对所有存活敌人施加 3 层萃血，并获得 8 点格挡。
- 恰好半血执行低血分支。

高血分支承担稀有启动锚点；低血分支承担回复、铺设和即时防守，不额外返能。升级仍只降低费用。

#### 赤色余温

保持 1 费，始终回复 6 / 8 点生命。

- 若本回合失去过生命或打出前处于低血，获得 6 / 8 点格挡。
- 否则在高血且本回合未失血时抽 1 张牌。

低血判断在治疗前快照，恰好半血获得格挡。

## Rare / Power 审查结论

| 卡牌 | 最终方案 | 主要验收点 |
|---|---|---|
| 血色重构 `BloodRebuild` | 采用上文方案：高血调到半血、获 3 能量并抽 1；低血回复到半血、全体 3 萃血并获 8 格挡 | 50 / 26 / 25 / 12 生命，无半血空分支 |
| 血裔王庭 `BloodClanCourt` | 打出时立即对所有存活敌人施加 2 层失血，再获得现有能力 | 首回合即时价值与后续回合触发各一次 |
| 血魔形态 `BloodDemonForm` | 打出时立即获得 1 力量，再获得现有能力；不调整心烛数值 | 即时力量只结算一次，后续成长维持原逻辑 |
| 惜别终幕 `FarewellFinale` | 萃血、失血、心烛分别增加攻击次数，但总攻击次数最多 3 次；维持 2 费和 7 / 9 单次伤害 | 0 / 1 / 2 / 3 种状态分别验证，总次数不得超过 3 |
| 永续重盈 `EternalReplete` | 保持 2 费和现有能力；高血打出时抽 2，低血对所有存活敌人施加 2 层萃血并获得 1 血速 | 血魔重盈已触发后仍有即时收益，不再成为空牌 |
| 不死的血统 `ImmortalBloodline` | 低血打出后返 1 能量，其他效果与升级方式不变 | 25 生命返能、26 生命不返能，检查濒死保护循环 |
| 暴血乱舞 `BloodFrenzy` | 低血打出后返 1 能量，其他伤害与多段逻辑不变 | 多段、血速和力量组合下每次打出最多返 1 能量 |

Rare/Power 的加强用于补足启动和低血行动力。惜别终幕保留多状态收益，但以最多 3 次总攻击控制多段触发；永续重盈不再降费，而是通过高低血即时收益解决抽到空牌的问题。
## 暂不改动

### 基础牌

血刃、血幕、玫影觅迹、断续脉搏维持现状。起始牌组当前平均输出 `18.77/turn`，第一轮不直接抬高基础攻击和格挡。

### 普通牌

血爪连斩、吸血、血香、血速、血染、血雾、献血、血盾、溅红、血击、血涌、烛痕刺、凝血本能、深红鞭痕、深红壁垒、心烛印记、切创、蔷薇步、玫刺维持现状。

血涌保留“攻击后失去 3 点生命”的高风险特征，但必须加入专门测试，确认无濒死保护时是否允许自杀是明确设计而非遗漏；在结论前不贸然改成普通生命费用。

### 稀有牌

深红献祭、红烛万象、残烛复燃、灵与欲的惜别本轮维持现状。

心烛相关稀有牌只做回归，不主动回调数值。若后续实战仍有资源问题，再依据同种子边际对照追加加强，不以高血木桩理论值作为削弱理由。
## sim2 验收

详细矩阵见 `docs/superpowers/reviews/2026-07-10-entelechia-sim2-balance-matrix.md`。

第一轮实现前先扩展 sim2：

1. 支持玩家起始生命和每回合固定敌伤。
2. 记录有效回复、格挡、自伤、最终生命、死亡率、抽牌、出牌、获得能量。
3. 报告是否命中每回合 100 次出牌上限。
4. 对满血、26、25、12/13 生命分别测试。
5. 使用同种子单卡边际对照，不以 Best-of-K 最高伤害结果证明生存提升。

输出回归下限：

- 起始牌组不低于 `18.77/turn`。
- 萃血节奏不低于 `15.88/turn`。
- 血速多段不低于 `20.43/turn`。
- 心烛爆发不因非心烛改动下降超过 5%，当前参考为 `49.47/turn`。
- 惜别引擎应高于当前 `45.73/turn`，并确认提升来自惜别终幕实际打出。
- 能量循环不低于 `22.90/turn`。
- 低血续航不只比较当前 `15.11/turn` 伤害，还必须在固定敌伤下改善死亡率或平均最终生命。

以下编号与实施计划一致：

1. Task 1：扩展 sim2 玩家起始生命。

3. Task 3：建立 `hidden-full-balance` 全牌库基线命令。
4. Task 4：增加统一高/低血判断，实施血愈、自噬、血色重构和赤色余温。
5. Task 5：实施 Uncommon 第一批：血暴、借血、凝血回流、血债清算、血宴。
6. Task 6：实施 Uncommon 第二批：断魂采血、凝血屏障、血祭之礼、痛觉转化。
7. Task 7：实施 Uncommon 第三批：放血、血脉过载、剖血、绯红壁合。
8. Task 8：实施心烛支援批：血色横扫、熔血为烛、余烛；心烛仪式只做回归。

10. Task 10：运行完整构筑、生存、循环与心烛回归，记录结果。
